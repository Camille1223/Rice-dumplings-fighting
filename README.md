# 粽子龙舟大赛 🎋🐲

端午限定 · 浏览器小游戏。三只 Q 版小粽子坐着龙舟，和 AI 龙舟一较高下。

## 在线游玩

👉 https://camille1223.github.io/-/

## 玩法

- 倒计时 3、2、1、**划！**
- 交替按 **A / D**（或 ← / →）划桨
- 节奏越规律加速越高，连续完美会触发连击爆发
- 同一键连按 = 乱桨减速；间隔太长 = 脱拍
- 先到终点者胜

## 难度

| 档位 | 距离 | 时长 |
|------|------|------|
| 轻松 | 400m | ~30s |
| 常规 | 600m | ~45s |
| 拼命 | 800m | ~60s |

## 本地运行

直接双击 `index.html` 即可，无需任何依赖。
